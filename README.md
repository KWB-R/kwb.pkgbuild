# kwb.pkgbuild

Helper functions for automating R package development at KWB to a predefined 
style.

# Installation

```r
#install.packages("devtools", repos = "https://cloud.r-project.org")
devtools::install_github("KWB-R/kwb.pkgbuild")
```
